# Cylon Detector voice alignment

## Target register

Builder first, Anti-Hero second. Deep wisdom with sassy swagger. Evidence before posture. Both the scientific method and the messy human judgment stay visible.

## What changed

- Replaced the generic detector description with the product's true line: “The detector does not tell you what is conscious. It finds where a theory goes soft.”
- Replaced vague provider language with the actual OpenAI boundary and `store: false` behavior.
- Changed machine-like status messages into direct statements about what the model did, what the human owns, and what remains blocked.
- Named the Mind, AI & Consciousness Lab in the credit instead of hiding it inside “collaborators.”
- Kept the phrases that already carry the product: attack, bleed, last word, and portable receipt.
- Kept the scientific contract intact: no consciousness score, no model verdict, no claim that citation matching validates an interpretation.

## Guardrails

- Profanity is not a substitute for voice.
- Battle language should clarify the adversarial method, not turn every button into cosplay.
- The interface can be fun without pretending the alpha is validated science.
- Every swagger line needs a nearby receipt, limit, or human owner.
